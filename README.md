# Muniraju AjithKumar

Personal portfolio website for Muniraju AjithKumar. This repository contains a static HTML/CSS/JS site with pages for about, projects, publications, certifications, services, and trainer information.

Overview
- Static site built with plain HTML, CSS and JavaScript.
- Pages: `index.html`, `abouts.html`, `projects.html`, `publications.html`, `certifications.html`, `services.html`, `trainer.html`.

Local development
- Open `index.html` in a browser to view the site locally.
- For a simple local server (recommended), run:

```powershell
# from the repository root
python -m http.server 8000
# then open http://localhost:8000
```

Repository structure
- `css/` — stylesheets (contains `style.css`)
- `js/` — frontend JavaScript (`main.js`)
- `images/` — image assets
- `Muniraju-AjithKumar/` — additional project notes (README inside)
